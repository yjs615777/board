## 프로젝트 개요
Board - 여러 사용자들이 소통할 수 있는 게시판서비스

## 사용 기술
Backend
- Java 17
- Spring Boot 3
- Spring Security
- JWT
- Spring Data JPA
- MySQL (AWS RDS)
- Caddy (Reverse Proxy + HTTPS)
- EC2 Amazon Linux 2023
- systemd 서비스 등록 (JAR 운영)

Frontend
- React (Vite)
- React Router
- Axios
- Custom CSS
- Vercel (배포)
